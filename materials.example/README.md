# Materials Directory Example

Real study materials and PDF excerpts must be stored in `materials/`.

The `materials/` directory is ignored by Git because it may contain copyrighted books, private materials, or large files. Do not commit real PDFs to this repository.

Suggested local structure:

```text
materials/
  uz/
    tarix/
    matematika/
    ona_tili/
    chet_tili/
    ozbekiston_tarixi/
  ru/
    history/
    math/
    native_language/
    foreign_language/
    uzbekistan_history/
```

This `materials.example/` directory is only documentation. It is safe to commit because it does not contain real study materials.
